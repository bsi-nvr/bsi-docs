## Getting Started

To get started with our internal systems, make sure you have:
1. Registered an account on the dashboard.
2. Configured your Bitwarden Vault URL under **Global Integrations** if you use a self-hosted instance.

## Managing Credentials

Always use the **Bitwarden Item Link** fields in our **Structured Docs** to connect credentials. Do not write plaintext passwords in standard markdown articles.
